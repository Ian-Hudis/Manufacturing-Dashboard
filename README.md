# Manufacturing-Dashboard
Read Data from MTConnect and Production Floor PLC to log data into SQL Tables and store large amounts of machine data for machine tracking.
<img width="1336" height="917" alt="image" src="https://github.com/user-attachments/assets/4f686779-8f9e-4f97-ad1c-99a30ecdab60" />

It basically breaks down to 4 components:

1. The MTConnect Data grab
2. The Kiosk Data Grab
3. Displaying the data onto a dashboard (see below)
4. Logging the Data into SQL.
